## 🔄 Toggle Dev Mode

### Disable Dev Mode (Temporarily):
```powershell
# Just rename the file
Rename-Item .devmode .devmode.disabled

# Run app - beta checks will be active
.venv\Scripts\python.exe run.py
```

### Re-enable Dev Mode:
```powershell
# Rename back
Rename-Item .devmode.disabled .devmode

# Run app - dev mode active again
.venv\Scripts\python.exe run.py
```
