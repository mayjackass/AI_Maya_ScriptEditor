INSTALLATION (2 MINUTES)
------------------------

1. EXTRACT FILES
   Extract this ZIP to your Maya scripts folder:
   
   Windows: C:\Users\<YourName>\Documents\maya\scripts\ai_script_editor\
   Mac:     ~/Library/Preferences/Autodesk/maya/scripts/ai_script_editor/
   Linux:   ~/maya/scripts/ai_script_editor/


2. INSTALL PYSIDE6
   Open Command Prompt or Terminal:
   
   pip install PySide6
   
   OR use Maya's Python:
   
   mayapy -m pip install PySide6


3. SET API KEY (Optional - For Morpheus AI)
   Edit: ai/__init__.py
   
   Change this line:
   OPENAI_API_KEY = ""
   
   To:
   OPENAI_API_KEY = "your-key-here"
   
   Get your key from: https://platform.openai.com/api-keys


4. LAUNCH IN MAYA
   Open Maya Script Editor (Windows > General Editors > Script Editor)
   
   In Python tab, paste and run:
   
   import sys
   sys.path.insert(0, r"C:\Users\<YourName>\Documents\maya\scripts\ai_script_editor")
   from main_window import AiScriptEditor
   neo_window = AiScriptEditor()
   neo_window.show()


DONE! The NEO Script Editor window should appear!

TESTING THE INSTALLATION
-------------------------
1. Type some Python code in the editor:
   print("Hello from NEO!")
   import maya.cmds as cmds
   cmds.sphere()

2. Click Run or press Ctrl+Enter
3. A sphere should appear in Maya viewport

TROUBLESHOOTING
---------------
Issue: "ModuleNotFoundError: No module named 'PySide6'"
Fix:   mayapy -m pip install PySide6

Issue: "ImportError: cannot import name 'AiScriptEditor'"
Fix:   Check your path is correct in sys.path.insert()

Issue: "Morpheus AI not responding"
Fix:   1. Check API key in ai/__init__.py
       2. Verify internet connection
       3. Check OpenAI account has credits
